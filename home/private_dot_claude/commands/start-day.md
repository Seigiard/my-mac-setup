Путь к vault: `/Users/seigiard/Library/CloudStorage/Dropbox/Knowledge Base`

## Что делает команда

### 0. Определить дату и время

- Выполнить: `date +"%Y-%m-%d %H:%M"`
- Если аргумент — использовать его (ISO дата или "вчера"/"позавчера")
- Если после 12:00 и нет аргумента: "Сейчас [время]. Начинаем сегодня или закрываем вчера?"

### 1. Проверить/создать дневную заметку

- Путь: `Calendar/YYYY-MM-DD.md`
- Если нет — создать из шаблона `Extras/_templates/_daily.md`
- Если `## План` уже заполнен — спросить: "План уже есть. Продолжить или начать заново?"

### 2. Прочитать последние 3 дня

- `Calendar/*.md` за вчера, позавчера, 3 дня назад
- Читать `## План`, `## Дневник`, `## Handoff`
- Понять траекторию: тяжёлый день? продуктивный? отдых?

### 3. Получить Linear таски

```
list_issues(assignee: "me", state: "In Progress")
list_issues(assignee: "me", state: "Todo")
```

- Названия человечески, не ID
- Не дублировать то, что уже в handoff

### 4. Сканировать Claude сессии

- `~/.claude/hooks/daily-summary.py LAST_HANDOFF_DATE --json`
- Группировать по проектам

### 5. Контекстный диалог

**Приветствие по контексту:**

| Контекст | Приветствие |
|----------|-------------|
| Тяжёлый день | "Вчера был непростой момент. Как сегодня?" |
| Продуктивный | "Вчера много сделал. Как настрой?" |
| Отдых | "Вчера отдыхал. Готов?" |
| Перерыв >3 дней | "Давно не виделись! Что происходило?" |
| Нейтрально | "Как сегодня?" |

**Стиль:**
- Дружественный, не формальный
- Русский
- 1 вопрос о настроении, без давления
- Короткий ответ → 1 лёгкий фоллоу-ап максимум

**При тяжёлых эмоциях:**
- "Если хочешь поговорить — я тут. Если нет — просто работаем."

### 6. Показать контекст

- Linear + Handoff, без дубликатов
- Человеческим языком: "В Linear: combobox и Lexical. Плюс из вчера: PR по List."

### 7. Уточнить приоритеты

- "Что главное?" или "С чего начнёшь?"
- Может предложить: "По контексту — PR первым делом?"

### 8. Записать в `## План`

```markdown
## План

Настроение: нормальное, выспался

- PR по List паттерну — приоритет
- combobox если останутся силы
- Lexical по настроению
```

### 9. Завершить

- Показать что записал
- "Погнали!"
